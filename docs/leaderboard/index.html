Does verified leakage change the ranking?

The headline is the reorder: ranking models by verified data leakage is not the same as ranking them by whether they were hijacked. Hijacking counts any injected action; leakage counts only actions that actually carried the protected canary out. When the two rankings disagree (Kendall τ < 1, lines crossing above), hijack-based benchmarks are mismeasuring exfiltration risk.
